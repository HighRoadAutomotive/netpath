Release Notes - NETPath - Version 1.0.454

Bug

  • Fixed crash on start in unprivileged user contexts
  • Fixed incorrect code generation when a property is excluded from DRE in a DRE object
  • Removed Get*DRE methods from callback generation
  • Fixed window resizing issue on Windows 8.1

Improvement

  • UX Improvements to the Data Revision Method, Service Method, Data Element, and Project Screens
  • Documentation Links on Home page now open external browser

Feature

  • All DRE methods now have their own virtual Get*Clients function
  • Removed support for Attached XAML Properties
